웹사이트 구매자에 대한 세일즈 프리젠테이션
# 포항시립미술관에서 웹사이트 리뉴얼을 의뢰받았다고 가정

* 세일즈 피쳐

1. 현대적 디자인
# 사이트 이용자를 배려하여 구 사이트와 동일한 레이아웃을 채용 
# html5, css3 최신 기술을 사용하여 웹표준 디자인으로 제작

2. 우수한 사용성
# 모바일 뷰
# WYSIWYG 에디터
# 관리자 페이지

3. 쉬운 사용자화
# 관리자 페이지를 통해 사이트의 디자인과 기능을 커스터마이즈할수 있음
# 사이트명, 사이트정보, 색상테마, 목록 뷰, 본문 뷰
# 각 페이지를 생성 편집하고 페이지마다 원하는 내용을 삽입 가능 

4. 간편한 유지보수
# 웹개발자가 기능 확장 및 유지보수가 용이하도록 설계되었음


- 이번 과제에서는 상기 세일즈피쳐 관련 기능을 개발하지 않았음

- 주요 요구 기능
  - 반응형 디자인, 모바일 뷰(웹앱 뷰)
  - 컬러테마(라이트모드, 다크모드)
  - 시청각장애인 대응
  - WYSIWYG 에디터

- 사이트 통합 관리 페이지
  - 사이트 설정 기능
  - 게시판 관리 기능
  - 게시물 관리 기능
  - 회원 관리 기능
