일정: 13일 ~ 23일

-- 열흘이라고 하지만 작업량 최소화에 신경써야 함
-- 기존 개발했던 소스를 최대한 재사용

* 기본 설계
-- CMS (WordPress, XpressEngine) 와 동일한 구조
-- 페이지, 카테고리, 인덱스 등 리퀘스트를 받아 해당 페이지를 다이나믹하게 출력
-- .htaccess 통해 설정된 rewrite 모듈로 home/abc/123 주소를 home/main.php?page=abc&cat=123 식으로 변환
-- 글 내용은 WYSIWYG 에디터를 통해 입력된 html 형식

* 디자인
-- 메인페이지, 목록페이지, 본문페이지, 통합검색페이지로 분류.
-- 메인페이지는 컨텐츠에 최신글 리스트 및 바로가기 위젯이 배치된 구조
-- 목록페이지는 세가지 스타일(내용 미리보기, 사진 미리보기, 테이블리스트 보기) 을 가지고 있음
-- 본문은 html 형식. 주요 스타일은 미리 다 지정되어 있고 img, p, b, span 등 태그가 사용되어 있음
-- 일부 본문내에 테이블이 삽입되는 경우가 있는데 굳이 별도 DB를 쓸 필요는 없음.

* 예외사항
-- 검색은 사이트 내부에서 처리해 본문에서 결과 표시하는걸로 디자인함. (통합검색과 게시판 검색을 따로 처리)
-- 대부분 컨텐츠는 새로 만들지 않고 소스 텍스트를 카피해서 html 로 작성함
-- 외부 페이지 링크는 따로 디자인하지 않고 그냥 해당 url 링크를 검
-- 한영 전환, 모바일 모드, 다크모드는 따로 만들지 않음
-- 로그인은 팝업을 띄워서 처리. 로그인 성공하면 메인페이지가 아니라 현재 페이지로 돌아감
